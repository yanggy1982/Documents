
# ollama使用

## 一、常用命令

ollama serve         #启动ollama  
ollama create        #从模型文件创建模型  
ollama show          #显示模型信息  
ollama run           #运行模型  
ollama pull          #从注册表中拉取模型  
ollama push          #将模型推送到注册表  
ollama list          #列出模型  
ollama cp            #复制模型  
ollama rm            #删除模型  
ollama help          #获取有关任何命令的帮助信息

## 二、下载Embedding模型

### 1、bge-large-zh-v1.5

ollama pull quentinz/bge-large-zh-v1.5

journalctl -u ollama -f
