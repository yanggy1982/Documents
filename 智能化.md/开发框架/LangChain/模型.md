## 2、大语言模型的能力

- **文本生成**
- **工具调用**：调用外部工具（如数据库查询或API调用）并在其响应中使用结果。
- **结构化输出**：模型的响应被限制在遵循定义的格式。
- **多模态**： 处理和返回除文本之外的数据，如图像、音频和视频。
- **推理**：模型通过多步骤推理得出结论。

## 3、模型调用

### 3.1、方式一：统一接口init_chat_model

例如：

```python
from langchain.chat_models import init_chat_model

model = init_chat_model("gpt-5.1")
response = model.invoke("Why do parrots talk?")
print(response.text)
```
