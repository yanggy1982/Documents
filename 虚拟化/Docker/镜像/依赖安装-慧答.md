
# 依赖安装

## 1、FastAPI依赖包

```shell
pip install fastapi

pip install uvicorn

pip install gunicorn

pip install sse-starlette

# 后续可替换
pip install dependency-injector

pip install python-dotenv

pip install loguru

pip install pydantic_settings
```
