
# 自制基础镜像

## 一、下载基础镜像

### 1、官方 

```shell
docker pull continuumio/miniconda3
```

## 二、创建并启动容器

```shell
docker run -it --name="smart-writing-base" continuumio/miniconda3 /bin/bash

docker run -it --name="assistant-ai-base" continuumio/miniconda3 /bin/bash

```

## 三、创建conda环境，并安装所需要的包

```shell
conda create -n smart-writing python=3.12

conda activate smart-writing

pip install packagename

```

## 四、将容器生成镜像

```shell
docker commit -a "king" smart-writing-base smart-writing-base:v1

docker save -o smart-writing-base-v1.tar smart-writing-base:v1 

docker commit -a "king" assistant-ai-base assistant-ai-base:v1

docker save -o assistant-ai-base-v1.tar assistant-ai-base:v1 


```
