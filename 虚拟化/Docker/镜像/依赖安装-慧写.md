
# 依赖安装

## 1、FastAPI依赖包

```shell
pip install fastapi

pip install uvicorn

pip install gunicorn

pip install sse-starlette

# 后续可替换
pip install dependency-injector

pip install python-dotenv

pip install loguru

pip install pydantic_settings
```

## 2、数据库依赖

```shell
pip install pymysql

pip install aiomysql

```

## 3、智能体依赖

```shell
pip install langgraph

pip install langchain-ollama

```
