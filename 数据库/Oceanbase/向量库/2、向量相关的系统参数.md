
# 向量相关的系统参数

## 1、向量索引内存占租户内存的比例（ob_vector_memory_limit_percentage）

```sql
-- 配置向量索引内存占租户内存的比例
-- V4.4.1 版本之前 默认30
-- V4.4.1 版本之后 默认值为 0 表示自适应模式，即系统自动调整租户中的向量索引数据的内存占用比例，无需手动调整：
-- 租户实际内存为 8GB 及以下时，该值自适应为 40。
-- 租户实际内存为 8GB 以上时，该值自适应为 50。
show parameters like 'ob_vector_memory_limit_percentage';
```
