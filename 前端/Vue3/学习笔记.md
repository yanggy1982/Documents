### 2.1、创建方法

- **vue-cli**

### 2.2、vue-cli创建

过程如下：

```shell
# 查看vue版本
vue --version

# 安装或者升级你的@vue/cli
vue install -g @vue/cli

# 项目创建
vue create vue_test
```

### 2.3、vite创建

#### 2.3.1、vite简介

vite是新一代前端构建工具，官网地址：https://vitejs.cn，vite的优势如下：

- 轻量快速的热重载（HMR），能实现快速的服务启动。
- 对 TypeScript、JSX、CSS等支持开箱即用。
- 真正的按需编译，不再等待整个应用编译完成。
